# Mein Lerntagebuch 📓

**7 Tage Vibe Coding Bootcamp — dokumentiert in Git.**

Ich bin Alex Heyers, Teilnehmer des Vibe Coding Bootcamps (Digitale Leute School, Kohorte 05/26).
Kein Code-Hintergrund, dafür 20 Jahre Berufserfahrung und die feste Absicht, das hier öffentlich durchzuziehen — Learning in Public.

## Worum geht es hier?

Dieses Repository ist eine Übung aus **Modul 2 — Technical Foundations**:
Sieben Tage lang jeden Tag ein Eintrag. Was habe ich gelernt? Welches Problem hatte ich? Wie habe ich es gelöst?

Ein Commit pro Tag. Ein Feature-Branch. Ein Pull Request. So wie man es im echten Leben macht.
